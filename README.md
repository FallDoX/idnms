# Trip Log Analyzer

Кроссплатформенный инструмент для анализа телеметрии поездок из CSV логов. Работает на **Windows**, **macOS** и **Linux** в любом браузере.

## Быстрый старт

### Вариант 1: Готовая сборка (рекомендуется)

Если у вас есть собранная версия в папке `dist`:

**Windows:**
```cmd
# Откройте index.html в браузере двойным кликом
# ИЛИ запустите локальный сервер:
start.bat
```

**macOS / Linux:**
```bash
# Запустите локальный сервер:
./start.sh
# Или откройте index.html в браузере
```

### Вариант 2: Режим разработки

1. Убедитесь что установлен [Node.js](https://nodejs.org/) (версия 18+)
2. Откройте терминал в папке `log-analyzer`
3. Выполните:
   ```bash
   npm install
   npm run dev
   ```
4. Откройте ссылку в браузере (обычно `http://localhost:5173`)
5. Перетащите ваш CSV файл в окно браузера

### Вариант 3: Статическая сборка

```bash
npm run build
```

После сборки откройте `dist/index.html` в браузере или запустите:
```bash
npm run serve
```

## Использование

1. **Загрузка файла:** Перетащите CSV файл в окно приложения или нажмите кнопку "Upload Trip CSV"
2. **Анализ данных:** Просматривайте графики и статистику
3. **Фильтрация по времени:** Используйте шкалу времени для выбора диапазона
4. **Настройка отображения:** Включайте/отключайте метрики и графики через панель настроек

Для подробного руководства по анализу ускорения см. [Руководство по анализу ускорения](docs/acceleration-user-guide.md)

## Возможности

### Анализ телеметрии
- **Карточки статистики:** Максимальная скорость, дистанция, пиковая мощность, расход заряда батареи и др.
- **Интерактивные графики:**
  - Профиль скорости (Speed vs GPS Speed)
  - Мощность и сила тока
  - Напряжение и уровень заряда (%)
  - Температура системы
- **Шкала времени:** Фильтрация данных по выбранному временному диапазону
- **Настройки отображения:** Включение/отключение метрик и графиков
- **Drag-and-Drop:** Удобная загрузка файлов

### Анализ ускорения
- **Автоматическое определение:** Детекция разгона по данным телеметрии
- **Настраиваемые пороги:** Конфигурация диапазонов скорости (0-60, 0-90, 0-100 км/ч и кастомные)
- **Таблица попыток:** Подробная информация о каждой попытке разгона
  - Время разгона
  - Дистанция
  - Мощность (средняя и пиковая)
  - Ток и напряжение
  - Падение батареи
  - Температура
- **Фильтрация:** Отображение только завершённых попыток
- **Выбор колонок:** Пользователь может выбрать какие метрики отображать
- **Сортировка:** Сортировка по любой метрике

### Режим сравнения
- **Выбор попыток:** Отметьте несколько попыток для сравнения
- **Наложенные графики:** Визуализация кривых скорости на одном графике
- **Таблица дельта-метрик:** Сравнение с лучшей попыткой
  - Δ Время
  - Δ Дистанция
  - Δ Мощность
  - Δ Батарея
- **Фильтрация:** "Лучшие N" и "Худшие N" попыток
- **Кривая мощности:** Отображение мощности на отдельной оси
- **Настраиваемый лимит:** Настройка количества отображаемых попыток

### Расширенные метрики
- **Эффективность мощности:** Мощность на единицу прироста скорости (Вт/(км/ч))
- **Согласованность мощности:** Стандартное отклонение мощности (0-1)
- **Распределение мощности:** Процент времени в диапазонах (<1000W, 1000-2000W, >2000W)
- **Падение батареи в секунду:** Скорость разряда (%/с)
- **Энергия на км:** Расход энергии (Wh/km)
- **Корреляция температура-мощность:** Связь между температурой и мощностью
- **Эффективность температуры:** Нормализованный показатель (оптимальный диапазон 20-35°C)
